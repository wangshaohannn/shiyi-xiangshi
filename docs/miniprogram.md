# 拾壹香室微信小程序

这是根据当前官网内容整理出的独立微信小程序项目，可直接用微信开发者工具打开 `miniprogram` 文件夹预览。

## 页面

- 首页：品牌首屏、本季香单、服务入口、札记。
- 香品：分类浏览、商品详情、咨询清单、本机订单记录。
- 调香：香材比例滑杆、场景预设、香方保存。
- 预约：体验项目、人数、日期、时段与联系方式提交。
- 我的：微信咨询、门店信息、本机订单、预约与香方记录。

## 打开方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择 `miniprogram` 文件夹。
4. AppID 可先使用测试号或游客模式，后续替换为正式 AppID。

当前版本使用本机缓存保存订单、预约和香方，方便演示。正式上线时可以把提交逻辑接到官网已有的 `/api/orders`、`/api/bookings` 和 `/api/formulas` 接口。
