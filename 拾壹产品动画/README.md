# 拾壹产品动画

这是一个独立的 3D 动态产品展示页面，用于展示拾壹香室沉香线香系列。

## 本地运行

```bash
cd /Users/gaohaoran/Documents/拾壹香室/拾壹产品动画
npm run dev
```

## 构建

```bash
cd /Users/gaohaoran/Documents/拾壹香室/拾壹产品动画
npm run build
```

## 设计方向

- 首屏使用 Three.js 制作沉香线香礼盒、香支、香盘和烟气动画。
- 视觉语言为东方极简：墨色主体、朱砂强调、克制金属微光。
- 产品内容来自 `沉香（线香）.docx`，图片来自现有 `assets/docx-product`。
